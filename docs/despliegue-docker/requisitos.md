# Requisitos

Antes de desplegar **CronoSENA** mediante Docker, asegúrate de contar con las siguientes herramientas instaladas en tu sistema:

## Herramientas necesarias

- **Git**: Para clonar el repositorio oficial de CronoSENA.
- **Docker**: Para la gestión y ejecución de los contenedores de la aplicación.
- **Docker Compose**: Para orquestar y gestionar los diferentes servicios de la aplicación de forma sencilla.
- **(Opcional en Windows) WSL2**: Recomendado para un mejor rendimiento de Docker en sistemas Windows.

## Instalación de herramientas

- **Git**: Descárgalo e instálalo desde [git-scm.com](https://git-scm.com/downloads).
- **Docker y Docker Compose**: Descárgalos desde [docker.com](https://www.docker.com/products/docker-desktop).
    - Docker Compose ya viene integrado en Docker Desktop en las versiones recientes.
- **WSL2 (solo Windows)**: Puedes seguir la guía oficial de Microsoft para habilitar WSL2: [Guía de instalación WSL2](https://docs.microsoft.com/es-es/windows/wsl/install).

> Asegúrate de que todas las herramientas estén correctamente instaladas y configuradas antes de continuar con el despliegue del sistema.
